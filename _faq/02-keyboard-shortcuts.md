---
title: What shortcuts are available?
---

| Keys          | Shortcut                                      |
|:-------------:| --------------------------------------------- |
| `↹`          | switch to next field                          |
| `⌘ ,`        | open the preferences panel                     |
| `⌘ o`        | add new PDF documents                          |
| `⌘ s`        | save the current document in your archive      |
| `⌘ ⌫`        | trash the selected document                   |
| `⌘ 9`        | zoom to fit                                    |
| `⌘ 0`        | zoom to original size                          |
| `⌘ -`        | zoom out                                       |
| `⌘ +`        | zoom in                                        |
